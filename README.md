Текст Священного Писания Нового Завета Господа нашего Иисуса Христа на церковнославянском языке не полностью соответсвует Textus Receptus или Патриархальному изданию 1904 г. Было принято решение сверстать две книги: с греческим текстом Textus Receptus и Патриархального издания 1904 г. Можно читать одну из книг, а в случае расхождения текстов на на церковнославянском и греческом языке, заглянуть в другую.
